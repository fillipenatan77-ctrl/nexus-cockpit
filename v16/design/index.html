6 · 5 princípios

1 · densidade pedagógica antes de UI

O hub serve cognição · não chamada de atenção. Hierarquia favorece conteúdo (citação, obra, ficha) sobre crômo gráfico.

2 · paleta dual estável

Dois eixos: ink (noite estoica) + gold (ouro contemplativo). Nunca introduzir terceiro eixo cromático sem revisão arquitetural.

3 · tipografia como filosofia

Georgia para conteúdo (densidade serif aristotélica) · SF Mono para metadados/UI (precisão monospace) · system sans para inputs.

4 · mobile-first iPhone

Safe-area sempre · touch targets ≥ 44pt · viewport-fit cover · backdrop-filter blur iOS-style. Desktop é fallback.

5 · offline-first

Conteúdo principal (cardápio, citações, obra do dia) inline no JS · zero fetch crítico. SW só amplia (não é dependência).

7 · usar com Claude Design

Claude Design (claude.ai/design · Anthropic Labs · lançado 17 abr 2026 · Claude Opus 4.7) cria design systems automáticos a partir de repos GitHub e Figma. Para usar com NEXUS:

  1. Abra claude.ai/design · plano Pro/Max/Team/Enterprise
  2. Importe o repo GitHub: fillipenatan77-ctrl/nexus-cockpit
  3. Claude analisa v16/index.html + esta página + cards SVG e infere o design system
  4. Use Claude Design para gerar novas páginas (e.g. landing page institucional, slides) usando as mesmas tokens
  5. Output volta ao repo via PR · merge mantém consistência visual NEXUS
