como abastecer · 5 passos
i
Abra Claude iOS · Project NEXUS Cortex
App Claude · menu lateral · Projects · selecione NEXUS Cortex. Se não existe ainda, crie · nome NEXUS Cortex.
ii
Configurações do Project → Conhecimento
Toque no ⚙️ ou ... ao lado do nome do projeto · selecione Knowledge ou Conhecimento do Projeto.
iii
Upload os 5 essenciais
Toque + Adicionar arquivo · faça upload um por um dos 5 arquivos marcados como essencial acima · pode ser via Safari (download primeiro) ou diretamente do iCloud Drive.
iv
Cole System Prompt v6.7 nas Custom Instructions
Voltar ao Project · Custom Instructions · cole conteúdo completo de nexuscaldeira.com/prompt-v67.md · save. Importante: mesmo o prompt já uploadado no Knowledge, ele PRECISA estar nas Custom Instructions também (lugares diferentes).
v
Validar · pergunta de probe
Em qualquer chat dentro do Project, pergunte: "Qual é a regra 11?" · resposta esperada inclui "Frente fechada = probe 🟢 + Scheduled Task ativa + GUIA_iOS atualizado". Se ele responder genericamente, repetir passo (iv) com prompt completo.

Não precisa sincronizar Drive com o Project · NEXUS Drive Reader / Make / IFTTT · qualquer fluxo automático.
