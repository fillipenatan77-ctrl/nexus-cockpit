# NEXUS Cockpit

App web PWA pessoal · cockpit do projeto NEXUS (PNL+IE v4.0 + PRIMORDIA).

**Live:** https://fillipenatan77-ctrl.github.io/nexus-cockpit/

**Instalar no iPhone:**
1. Abrir URL no Safari
2. Compartilhar ⬆️ → Adicionar à Tela de Início
3. Ícone N dourado aparece · funciona offline após 1º load

Stack: HTML single-page + Tailwind CDN + Alpine.js + PWA service worker (cache offline-first).

Código principal (privado): https://github.com/fillipenatan77-ctrl/NEXUS
