1 · Calendário PRIMORDIA no iPhone

iCal subscribível · zero manutenção · lembretes automáticos
Gerei um calendário com 98 eventos cobrindo todo o ciclo PRIMORDIA: início e fim, as 18 estações do Palácio em sequência, lembrete semanal de /revisao aos domingos, e da lente Memorização às sextas. Subscreva uma vez · atualiza sozinho.
1.1 No iPhone, abra o Safari e vá em: fillipenatan77-ctrl.github.io/nexus-cockpit/home/nexus-primordia.ics
1.2 iOS pergunta: "Abrir no Calendário?" → toque em Abrir
1.3 Toque em Adicionar Todos ou Inscrever (se oferecido)
1.4 Escolha o calendário: crie um novo "NEXUS" com cor dourada para separar dos demais
1.5 Em Configurações → Calendário → Contas → Assinaturas: confirme que está atualizando
⟶ baixar/subscrever calendário
Dica · ative "Alertar" em Configurações do calendário NEXUS para 1h antes de cada evento diurno — você recebe notificação lembrando da estação atual e do ritual semanal.

2 · Apple Shortcuts · Siri integrada

comandos de voz · acesso instantâneo sem abrir app
Crie atalhos simples que abrem páginas do hub · consultam o Apps Script · disparam uma surpresa. Depois é só dizer "E aí Siri, NEXUS hoje" e aparece o briefing do dia.
2.1 · "NEXUS Hoje"
Abra app Atalhos → + → Ação: Abrir URL → cole fillipenatan77-ctrl.github.io/nexus-cockpit/home/hoje.html → nomeie "NEXUS hoje" · ative "Adicionar à Siri"
2.2 · "NEXUS Surpresa" (dispara surprise no Apps Script)
Novo atalho → Obter Conteúdo de URL → cole script.google.com/macros/s/AKfycby76YrIrYocLMDN7VLnCtVcpVz3kquK6e5Q0gx5wwQQQQly3OQ9ZLlbT8sP0JNl8ufc/exec?action=surprise_now → Mostrar Alerta com o texto da resposta · nomeie "NEXUS me surpreenda"
2.3 · "NEXUS Status" (ciclo do dia falado)
Novo atalho → Obter URL → ...?action=retomada_status → Obter Valor de Dicionário (chave: cycle) → Falar Texto → "Siri, NEXUS status" · ela fala o dia/estação/lente em áudio
2.4 · "NEXUS Busca" (RAG via voz)
Atalho → Ditar Texto → Obter URL → broader-hang-alto-subcommittee.trycloudflare.com/ask com body {"q": [texto ditado]} → Mostrar Resultado
Ao adicionar à Siri, grave a frase de ativação em português · funciona no trânsito sem tocar no telefone.

3 · Web Push · notificação real no Lock Screen

iOS 16.4+ · requer PWA instalada via "Add to Home Screen"
Notificações reais iOS chegam via Apple Push Notification Service. Pré-requisito: a PWA NEXUS precisa estar instalada (Safari → compartilhar → Adicionar à Tela de Início). Depois é autorizar a permissão uma vez · e passa a receber pushes quando o sistema envia.
3.1 Instale a PWA (se ainda não): abra o hub no Safari → botão compartilhar → Adicionar à Tela de Início
3.2 Abra a PWA instalada (não pelo Safari · tem que ser pelo ícone na tela inicial)
3.3 Toque no botão abaixo · iOS pergunta "Permitir Notificações?" → Permitir
3.4 Pronto · o service worker fica registrado · quando o Apps Script enviar push, aparece no Lock Screen mesmo com app fechado
Limitações honestas: iOS Web Push funciona só em PWA standalone (não no Safari normal) · exige iOS 16.4 ou superior · pode precisar de app server VAPID configurado no Apps Script (fase seguinte).

§ outras possibilidades · próximas ondas

  • Widget na tela inicial via Scriptable — app gratuito que lê JSON de URL e mostra widget ao lado dos apps nativos. Podemos expor health.json e ter dia/estação/lente vivos na tela inicial sem abrir nada.
  • Tag NFC — cole uma tag NFC barata no painel do carro / mesa de trabalho. Ao tocar o iPhone, abre direto /hoje. Integração zero-clique.
  • Live Activities (iOS 16+) — timer Pomodoro do hub ficaria visível no Lock Screen com progresso ativo · requer app nativo mas há frameworks PWA experimentais.
  • Focus Mode "NEXUS" — crie em Configurações → Foco → filtra apenas Claude + WhatsApp (surprises) + Mensagens. Ative ao ler/estudar para silenciar o resto.
  • Share Sheet Extension — bookmark JavaScript que envia texto selecionado de qualquer app pro Cortex · útil para reflexão sobre notícia/artigo.
  • Apple Watch (se tiver) — complicação custom mostrando dia do ciclo · toca para ver estação atual.
  • RSS privado — feed de surpresas + novidades para leitores como Feedly · roda no fundo.
  • Email newsletter dominical — Apps Script envia todo domingo 20h com resumo da semana + citação destacada + próximos eventos PRIMORDIA.
  • NFC + Voz combinados — tag NFC no carro → abre Shortcut que ativa modo voz Claude direto. Um toque, zero dedos.
